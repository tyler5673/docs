The [You.com API](https://docs.you.com) is a suite of tools designed to help developers ground LLM output in the most recent, most accurate, and most relevant web information.

## Setup

### Installation

```bash
pip install -U langchain-youdotcom
```

### Credentials

Get an API key at [you.com/platform/api-keys](https://you.com/platform/api-keys), then set it as an environment variable:

```python
import os

os.environ["YDC_API_KEY"] = "your-api-key"
```

It's also helpful (but not needed) to set up [LangSmith](https://smith.langchain.com/) for best-in-class observability:

```python
os.environ["LANGSMITH_TRACING"] = "true"
```

## Utility usage

`YouSearchAPIWrapper` provides direct access to the You.com search, contents, and research APIs and powers the retriever and tools under the hood.

```python
from langchain_youdotcom import YouSearchAPIWrapper

wrapper = YouSearchAPIWrapper(count=5, livecrawl="web")
```

### Search

```python
docs = wrapper.results("What is LangChain?")

print(docs[0].page_content[:300])
print(docs[0].metadata)
```

### Raw results

```python
response = wrapper.raw_results("latest AI news")

for result in response.results.web[:2]:
    print(result.title, result.url)
```

### Contents API

Fetch and extract content from URLs:

```python
pages = wrapper.contents(["https://python.langchain.com"])

print(pages[0].page_content[:500])
print(pages[0].metadata["url"])
```

### Research API

Get comprehensive, research-grade answers with cited sources:

```python
# formatted markdown answer with sources
text = wrapper.research_text("compare RAG vs fine-tuning for domain adaptation")
print(text)

# raw SDK response for programmatic access
response = wrapper.raw_research("compare RAG vs fine-tuning for domain adaptation")
print(response.output.content)
```

Control depth with `research_effort`:

```python
wrapper = YouSearchAPIWrapper(research_effort="deep")  # lite, standard, deep, exhaustive
```

### Async

Async variants are available for all methods:

```python
import asyncio

docs = asyncio.run(wrapper.results_async("What is LangChain?"))
pages = asyncio.run(wrapper.contents_async(["https://example.com"]))
text = asyncio.run(wrapper.research_text_async("explain quantum error correction"))
```

## Retriever usage

```python
from langchain_youdotcom import YouRetriever

retriever = YouRetriever(k=5)
docs = retriever.invoke("What is retrieval augmented generation?")

for doc in docs:
    print(doc.metadata["title"])
    print(doc.page_content[:200])
    print()
```

### Parameters

```python
retriever = YouRetriever(
    k=5,                          # max documents to return
    count=10,                     # max results per API section
    livecrawl="web",              # live-crawl mode: "web", "news", or "all"
    livecrawl_formats="markdown", # "html" or "markdown"
    country="US",
    freshness="week",             # "day", "week", "month", or "year"
    safesearch="moderate",        # "off", "moderate", or "strict"
)
```

## Chaining

```bash
pip install -U langchain-openai
```

### Invoke

```python
docs = retriever.invoke("What happened in AI this week?")
```

### Stream

```python
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnablePassthrough

llm = ChatOpenAI(model="gpt-4o-mini")
prompt = ChatPromptTemplate.from_template(
    """Answer based on the following context:

{context}

Question: {question}"""
)


def format_docs(docs):
    return "\n\n".join(doc.page_content for doc in docs)


chain = (
    {"context": retriever | format_docs, "question": RunnablePassthrough()}
    | prompt
    | llm
    | StrOutputParser()
)

for chunk in chain.stream("What is LangChain?"):
    print(chunk, end="", flush=True)
```

### Batch

```python
queries = ["What is LangChain?", "What is LangGraph?"]
results = retriever.batch(queries)

for docs in results:
    print(docs[0].metadata["title"])
```
