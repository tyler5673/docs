The [You.com API](https://docs.you.com/api-reference/) is a suite of tools designed to help developers ground the output of LLMs in the most recent, most accurate, most relevant information that may not have been included in their training dataset.

## Setup

The retriever lives in the `langchain-community` package.

You also need to set your You.com API key.

```bash
pip install -qU langchain-community
```

```python
import os

os.environ["YDC_API_KEY"] = ""

# For use in Chaining section
os.environ["OPENAI_API_KEY"] = ""
```

It's also helpful (but not needed) to set up [LangSmith](https://smith.langchain.com/) for best-in-class observability

```python
os.environ["LANGSMITH_TRACING"] = "true"
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass()
```

## Utility usage

```python
from langchain_community.utilities import YouSearchAPIWrapper

utility = YouSearchAPIWrapper(count=3)
utility
```

### Raw results

```python
response = utility.raw_results(query="What is the weather in NY")
hits = response["results"]["web"]

print(len(hits))
print(list(hits[0].keys()))
```

### Parsed results

```python
response = utility.results(query="What is the weather in NY")

print(len(response))
print(response[0].page_content[:200])
print(response[0].metadata.keys())
```

Each document's metadata includes:
- `url`: source page URL
- `title`: page title
- `description`: page description
- `thumbnail_url`: thumbnail image URL (if available)
- `favicon_url`: site favicon URL
- `page_age`: estimated page age

### Livecrawl (full page content)

Enable livecrawl to get the full page content instead of snippets:

```python
utility = YouSearchAPIWrapper(count=2, livecrawl="web", livecrawl_formats="markdown")
docs = utility.results("python programming language")

# Full page content instead of snippets
print(len(docs[0].page_content))
```

When livecrawl is enabled, each result returns one document with the full page content
rather than multiple snippet documents per page.

### Contents API

Extract content from specific URLs:

```python
utility = YouSearchAPIWrapper()
docs = utility.contents(
    urls=["https://en.wikipedia.org/wiki/Python_(programming_language)"],
    formats=["markdown"],
)

print(docs[0].page_content[:200])
```

## Retriever usage

```python
from langchain_community.retrievers import YouRetriever

retriever = YouRetriever(count=3)
docs = retriever.invoke("What is the weather in NY")

print(len(docs))
print(docs[0].metadata)
```

### With livecrawl

```python
retriever = YouRetriever(count=2, livecrawl="web", livecrawl_formats="markdown")
docs = retriever.invoke("python programming")

print(len(docs[0].page_content))
```

### Async support

```python
retriever = YouRetriever(count=3)
docs = await retriever.ainvoke("What is the weather in NY")
print(len(docs))
```

## Chaining

```bash
pip install -qU langchain-openai
```

```python
from langchain_community.retrievers import YouRetriever
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnablePassthrough
from langchain_openai import ChatOpenAI

runnable = RunnablePassthrough

retriever = YouRetriever(count=3)
model = ChatOpenAI(model="gpt-4o-mini")
output_parser = StrOutputParser()
```

### Invoke

```python
prompt = ChatPromptTemplate.from_template(
    """Answer the question based only on the context provided.

Context: {context}

Question: {question}"""
)

chain = (
    runnable.assign(context=(lambda x: x["question"]) | retriever)
    | prompt
    | model
    | output_parser
)

output = chain.invoke({"question": "what is the weather in NY today"})
print(output)
```

### Stream

```python
for s in chain.stream({"question": "what is the weather in NY today"}):
    print(s, end="", flush=True)
```

### Batch

```python
output = chain.batch(
    [
        {"question": "what is the weather in NY today"},
        {"question": "what is the weather in sf today"},
    ]
)

for o in output:
    print(o)
```
