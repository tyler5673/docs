The [You.com API](https://docs.you.com) provides LLM-optimized web search and content extraction, grounding model output in accurate, real-time web information.

## Setup

### Installation

```bash
pip install -U langchain-youdotcom
```

### Credentials

Get an API key at [you.com/platform/api-keys](https://you.com/platform/api-keys), then set it as an environment variable:

```python
import os

os.environ["YDC_API_KEY"] = "your-api-key"
```

It's also helpful (but not needed) to set up [LangSmith](https://smith.langchain.com/) for best-in-class observability:

```python
os.environ["LANGSMITH_TRACING"] = "true"
```

## YouSearchTool

Search the web with You.com and return results as formatted text.

```python
from langchain_youdotcom import YouSearchTool

tool = YouSearchTool()
result = tool.invoke("latest developments in AI agents")
print(result)
```

### With parameters

Pass search parameters via `api_wrapper`:

```python
from langchain_youdotcom import YouSearchAPIWrapper, YouSearchTool

tool = YouSearchTool(
    api_wrapper=YouSearchAPIWrapper(
        k=3,
        livecrawl="web",       # live-crawl mode: "web", "news", or "all"
        freshness="day",       # "day", "week", "month", or "year"
        safesearch="moderate", # "off", "moderate", or "strict"
    )
)
result = tool.invoke("AI news today")
```

## YouContentsTool

Fetch and extract content from web pages.

```python
from langchain_youdotcom import YouContentsTool

tool = YouContentsTool()
result = tool.invoke({"urls": ["https://python.langchain.com"]})
print(result[:500])
```

### Multiple URLs

```python
result = tool.invoke({
    "urls": [
        "https://python.langchain.com",
        "https://docs.you.com",
    ]
})
```

## Use within an agent

Both tools work with any LangChain agent. Here's an example using LangGraph's `create_react_agent`:

```bash
pip install -qU langgraph langchain-openai
```

```python
import os
from langchain_openai import ChatOpenAI
from langchain_youdotcom import YouSearchTool, YouContentsTool
from langgraph.prebuilt import create_react_agent

os.environ["OPENAI_API_KEY"] = "your-openai-api-key"

llm = ChatOpenAI(model="gpt-4o-mini")
tools = [YouSearchTool(), YouContentsTool()]
agent = create_react_agent(llm, tools)

response = agent.invoke(
    {"messages": [{"role": "user", "content": "What is the latest news about LangChain?"}]}
)
print(response["messages"][-1].content)
```
