The [You.com API](https://docs.you.com/api-reference/) is a suite of tools designed to help developers ground the output of LLMs in the most recent, most accurate, most relevant information that may not have been included in their training dataset.

## Setup

The tools live in the `langchain-community` package.

You also need to set your You.com API key.

```bash
pip install -qU langchain-community
```

```python
import os

os.environ["YDC_API_KEY"] = ""

# For use in Agent section
os.environ["OPENAI_API_KEY"] = ""
```

It's also helpful (but not needed) to set up [LangSmith](https://smith.langchain.com/) for best-in-class observability

```python
os.environ["LANGSMITH_TRACING"] = "true"
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass()
```

## YouSearchTool

Wraps the You.com Search API. An agent can use this tool to search the web for current information.

```python
from langchain_community.tools.you import YouSearchTool
from langchain_community.utilities.you import YouSearchAPIWrapper

api_wrapper = YouSearchAPIWrapper(count=3)
tool = YouSearchTool(api_wrapper=api_wrapper)
tool
```

```python
response = tool.invoke("What is the weather in NY")

print(len(response))
print(response[0].page_content[:200])
print(response[0].metadata.keys())
```

### With livecrawl

Get full page content instead of snippets:

```python
api_wrapper = YouSearchAPIWrapper(
    count=2, livecrawl="web", livecrawl_formats="markdown"
)
tool = YouSearchTool(api_wrapper=api_wrapper)

response = tool.invoke("python programming")
print(len(response[0].page_content))
```

## YouContentsTool

Wraps the You.com Contents API. An agent can use this tool to extract content from specific URLs.

```python
from langchain_community.tools.you import YouContentsTool
from langchain_community.utilities.you import YouSearchAPIWrapper

api_wrapper = YouSearchAPIWrapper()
tool = YouContentsTool(api_wrapper=api_wrapper)
tool
```

```python
response = tool.invoke(
    {"urls": ["https://en.wikipedia.org/wiki/Python_(programming_language)"]}
)

print(len(response))
print(response[0].page_content[:200])
```

### Async support

Both tools support async:

```python
response = await tool.ainvoke(
    {"urls": ["https://en.wikipedia.org/wiki/Python_(programming_language)"]}
)
```

## Use within an agent

```bash
pip install -qU langchain langchain-openai langgraph langchain-community
```

```python
from langchain_community.tools.you import YouSearchTool, YouContentsTool
from langchain_community.utilities.you import YouSearchAPIWrapper
from langchain_openai import ChatOpenAI
from langgraph.prebuilt import create_react_agent

api_wrapper = YouSearchAPIWrapper(count=3)
search_tool = YouSearchTool(api_wrapper=api_wrapper)
contents_tool = YouContentsTool(api_wrapper=api_wrapper)

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

agent = create_react_agent(llm, [search_tool, contents_tool])

result = agent.invoke(
    {"messages": [{"role": "user", "content": "What is the weather in NY today?"}]}
)

for message in result["messages"]:
    message.pretty_print()
```
